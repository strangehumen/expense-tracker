# 支出管理アプリ

HTML、CSS、JavaScriptで作成したシンプルな支出管理アプリです。

## 制作背景

日々の支出を手軽に記録・管理できるアプリが欲しかったため制作しました。

スマホでも使いやすく、データをブラウザ内に保存できるため、インストール不要で利用できます。

## 主な機能

- 支出の追加
- 支出の編集
- 支出の削除
- カテゴリー別管理
- キーワード検索
- 並び替え（新しい順、古い順、金額順）
- 月間合計金額の表示
- カテゴリー別集計
- 円グラフによる支出の可視化
- CSVエクスポート
- CSVインポート
- ダークモード
- localStorageによるデータ保存

## 使用技術

- HTML
- CSS
- JavaScript
- Chart.js
- localStorage

## 特徴

- レスポンシブデザインに対応
- スマートフォンでも快適に利用可能
- データはブラウザ内に保存されるためサーバー不要
- シンプルで直感的なUI
